A multi-agent framework built to **run on device**: small protocols you extend, swappable agents,
classifier routing, and pluggable chat storage — plus what an on-device assistant needs.

:::caution[Work in progress]
Built incrementally, one reviewed component at a time. The public API is not yet stable.
:::

## Requirements

- **iOS 16+** / **macOS 14+**
- **Swift 6.2** toolchain (Xcode 26+), Swift 6 language mode

The whole framework runs on iOS 16+, including the bundled `FileChatStorage`. The one piece gated
higher is `DeviceChatStorage` (SwiftData), which needs iOS 17+ / macOS 14+.
